# Retail Sales Performance Dashboard (Excel & Power BI)

This project analyzes retail sales performance using the Sample Superstore dataset.  
The analysis was implemented using both **Microsoft Excel** and **Power BI** to demonstrate data analysis, visualization, and dashboard design skills.

## Project Overview
The dashboard provides insights into sales performance, customer contribution, and category-level trends. It helps identify high-performing product categories, top customers, and sales patterns over time.

## Key Insights
- Total Sales, Profit, and Order volume tracking
- Sales performance across product categories
- Monthly sales trend analysis
- Top 10 customers by revenue
- Profit vs Sales comparison by category

## Dashboard Features
- KPI metrics (Total Sales, Total Profit, Total Orders)
- Sales by Category visualization
- Monthly Sales Trend line chart
- Top 10 Customers by Sales
- Interactive filters for Region and Category

## Tools Used
- **Microsoft Excel**
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Dashboard layout design

- **Power BI**
  - Interactive dashboards
  - Visualizations (bar charts, line charts, KPI cards)
  - Top N filtering
  - Slicers for dynamic filtering

## Dataset
Sample Superstore dataset containing retail transaction data including:
- Orders
- Customers
- Product categories
- Sales and profit information
