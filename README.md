# Retail Sales Performance Dashboard (Excel)

This project presents a retail sales analytics dashboard built using Microsoft Excel.

## Overview
The dashboard analyzes sales performance using the Sample Superstore dataset. 
It highlights key metrics and trends to support business insights.

## Features
- KPI tracking (Total Sales, Total Profit, Total Orders)
- Monthly sales trend analysis
- Top 10 customers by sales
- Category-level sales comparison
- Profit vs Sales analysis
- Interactive filtering using slicers (Region and Category)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Basic data analysis
